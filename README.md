# LocalMD - Markdown 编辑器

一款类似 Typora 的本地 Markdown 编辑器，仅支持 Windows 系统。

## ✨ 主要功能

- 📝 **实时预览** - 左侧编辑，右侧实时预览
- 💾 **文件管理** - 新建、打开、保存、另存为
- ✨ **AI 润色** - 接入 DeepSeek 大模型，智能润色文本
- 🎨 **主题切换** - 亮色/暗色主题，保存偏好设置
- ⌨️ **快捷键支持** - 常用 Markdown 快捷键
- 🛠️ **工具栏** - 快速插入标题、加粗、斜体、链接等
- 📋 **自动保存** - 自动保存草稿到本地
- 🔄 **同步滚动** - 编辑器和预览区域同步滚动

## 🚀 快速开始

### 安装依赖

```bash
npm install
```

### 运行应用

```bash
npm start
```

### 打包应用

```bash
npm run build
```

打包后的应用会生成在 `dist` 目录下。

## ⌨️ 快捷键

### 文件操作
- `Ctrl+N` - 新建文件
- `Ctrl+O` - 打开文件
- `Ctrl+S` - 保存文件
- `Ctrl+Shift+S` - 另存为

### 编辑操作
- `Ctrl+B` - 加粗
- `Ctrl+I` - 斜体
- `Ctrl+Shift+P` - AI 润色
- `Ctrl+Z` - 撤销
- `Ctrl+Y` - 重做
- `Tab` - 插入4个空格

### 视图操作
- `Ctrl+T` - 切换主题
- `F11` - 全屏
- `F12` - 开发者工具

## 🎯 支持的 Markdown 语法

- 标题 (H1-H6)
- 加粗、斜体、删除线
- 引用
- 代码块和行内代码
- 链接和图片
- 有序列表和无序列表
- 任务列表
- 表格
- 分割线

- **Electron** - 跨平台桌面应用框架
- **Marked** - Markdown 解析库
- **原生 JavaScript** - 无额外框架依赖

## 📝 使用说明

1. 启动应用后，可以直接在左侧编辑区开始输入 Markdown 内容
2. 右侧会实时预览渲染效果
3. 使用工具栏快速插入常用 Markdown 格式
4. 通过菜单栏或快捷键保存文件
5. 支持主题切换，设置会自动保存

### 🤖 AI 润色功能

1. **配置 API**
   - 点击右上角的“⚙️ 设置”按钮
   - 在 DeepSeek 官网（https://platform.deepseek.com）注册并获取 API Key
   - 将 API Key 填入设置界面
   - 可选配置：模型、Temperature、最大 Tokens、系统提示词
   - 点击“测试连接”验证配置是否正确
   - 点击“保存设置”

2. **使用 AI 润色**
   - 选中要润色的文本（或不选中，默认润色全文）
   - 点击工具栏的“✨ AI润色”按钮，或按 `Ctrl+Shift+P`
   - 等待 AI 处理完成，润色后的文本会自动替换原文

3. **AI 参数说明**
   - **模型**：
     - `deepseek-chat`：通用对话模型，适合文本润色
     - `deepseek-coder`：代码专用模型，适合润色技术文档
   - **Temperature**：控制创造性
     - 0.0-0.3：更保守，更精准
     - 0.4-0.7：均衡（推荐）
     - 0.8-1.0：更有创意
   - **最大 Tokens**：控制输出长度，默认 2000
   - **系统提示词**：自定义 AI 的行为风格
