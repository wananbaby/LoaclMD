## 第二步：添加 AI 润色功能

### 提示词 2.1 - 集成 DeepSeek AI

```
增加AI润色功能，接入deepseek大模型，deep seek参数设置独立
```

**AI 会做什么：**
- 创建 `deepseek.js`（AI 配置模块）
- 在界面上添加"AI 润色"按钮
- 添加设置界面，用于配置：
  - DeepSeek API Key
  - 模型选择
  - Temperature（创造性参数）
  - 最大 Tokens
  - 系统提示词
- 添加加载动画和错误提示

**完成后，你会得到：**
- ✅ AI 润色按钮（工具栏右侧）
- ✅ 设置按钮（⚙️ 图标）
- ✅ 完整的 AI 配置界面
- ✅ 快捷键 `Ctrl+Shift+P` 触发 AI 润色

**如何使用 AI 润色功能：**

1. 获取 DeepSeek API Key：
   - 访问 https://platform.deepseek.com
   - 注册账号
   - 进入控制台，创建 API Key
   - 复制 API Key

2. 配置应用：
   - 点击编辑器右上角的 ⚙️ 按钮
   - 粘贴你的 API Key
   - 点击"测试连接"验证
   - 点击"保存设置"

3. 开始使用：
   - 在编辑器中选中要润色的文本
   - 点击"✨ AI润色"按钮
   - 等待 AI 处理，润色结果会自动替换原文

## 常见问题

### Q1: 提示"npm 不是内部或外部命令"

**解决方法：**
1. 确保已安装 Node.js
2. 重启电脑
3. 在命令行输入 `node -v` 检查是否安装成功

### Q2: AI 润色功能不可用

**解决方法：**
1. 确保已配置 DeepSeek API Key
2. 检查网络连接
3. 点击设置中的"测试连接"按钮验证

### Q3: 打包失败，提示权限不足

**解决方法：**
1. 右键点击 `build-installer.ps1`
2. 选择"使用 PowerShell 运行"
3. 在弹出的 UAC 窗口中点击"是"

### Q4: 安装包太大

**原因：**
- Electron 应用包含了完整的浏览器引擎
- 正常大小约 70-100MB

**优化方法：**
- 使用免安装版（portable）
- 压缩后分发

### Q5: 想修改 AI 提示词

**解决方法：**
1. 打开设置界面
2. 修改"系统提示词"字段
3. 保存设置即可
